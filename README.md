# Biopython-for-Bioinformatics
A practical Biopython tutorial repository covering biological sequence analysis, file handling, NCBI databases, BLAST, protein analysis and next-generation sequencing workflows.

# Repository Contents

1. Seq Objects
2. FASTA File Handling
3. GenBank File Handling
4. Basics of Entrez
5. Pairwise Alignment
6. Protein Analysis
7. FASTQ Quality Control
8. BLAST with Biopython

- FASTQ quality assessment
- BLAST searches
- XML parsing
- CSV export
